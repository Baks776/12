# Быстрый старт

## Минимальная настройка за 5 минут

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Создание бота в Telegram
1. Откройте [@BotFather](https://t.me/BotFather)
2. Отправьте `/newbot`
3. Следуйте инструкциям и скопируйте токен

### 3. Узнайте свой Telegram ID
1. Откройте [@userinfobot](https://t.me/userinfobot)
2. Отправьте любое сообщение
3. Скопируйте ваш ID (число вида `123456789`)

### 4. Создайте файл `.env`
```env
TELEGRAM_BOT_TOKEN=ваш_токен_от_BotFather
ADMIN_IDS=ваш_telegram_id
```

### 5. Запустите бота
```bash
python main.py
```

### 6. Проверьте работу
1. Откройте Telegram и найдите вашего бота
2. Отправьте `/start`
3. Отправьте `/chat_id` чтобы увидеть ID чата

### 7. Проверьте работу
1. Откройте Telegram и найдите вашего бота
2. Отправьте `/start`
3. Отправьте `/chat_id` чтобы увидеть ID чата

### 8. Создайте первую задачу

**Для текущего чата:**
```
/add_task 09:00 Доброе утро!
```

**Для конкретной группы:**
1. Добавьте бота в группу
2. Отправьте `/chat_id` в группе, чтобы узнать ID
3. Создайте задачу с указанием ID группы:
```
/add_task 09:00 Доброе утро! -1001234567890
```

Готово! Бот будет отправлять сообщение каждый день в 09:00.

## Полезные команды

- `/add_task 12:00 Обед пн,ср,пт` - сообщение в определенные дни
- `/add_task 09:00 Привет -1001234567890` - сообщение в конкретную группу
- `/list_tasks` - список всех задач
- `/delete_task <ID>` - удалить задачу
- `/add_admin <ID>` - добавить админа
- `/list_admins` - список админов
- `/chat_id` - узнать ID текущего чата/группы
